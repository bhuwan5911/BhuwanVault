## ğŸ“¦ How to Run

To run this project locally, you need a local web server (due to browser security restrictions on `file://` protocols for loading external project files).

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/bhuwan5911/100-Front-End-Projects.git
    cd 100-Front-End-Projects
    ```

2.  **Start a Local Server**
    *   **VS Code**: Right-click `index.html` and select **"Open with Live Server"**.
    *   **Node.js**:
        ```bash
        npx serve .
        ```
    *   **Python**:
        ```bash
        python -m http.server
        ```

3.  **Open in Browser**
    *   Navigate to `http://localhost:5500` (or your server's port).
